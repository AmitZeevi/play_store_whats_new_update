# play_store_whats_new_update

This Python script was created to prevent the manual work caused when one wishes to change it's 'What's new' section
in the google console for spaces and owner app.

## How to use?
1. run 'python translate.py'
2. Enter .xlsx file with the translated sections - same as in this folder
3. The translated files will be located in the folder 'files_to_upload'
